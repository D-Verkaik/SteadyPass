UPDATE DIRECTIONS

WARNING: Do not Unplug your board from the usb port during the update. 
In some cases this may cause the system to be unrecoverable.

1. Plug in a USB cable to your Steady Pass System and connect 
to a computer.
2. double click to run SteadyPassEZUpdate.bat
3. Wait for UPDATE COMPLETE message
4. Get back to the water!
